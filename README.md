# Hangman (Виселица)

Консольная игра «Виселица» с возможностью выбора уровня сложности и языка.

## Возможности

* Выбор уровня сложности: **легкий**, **средний**, **сложный**
* Выбор языка словаря: **английский** / **русский**
* Поддержка Linux и Windows
* Удобный запуск через `uv` или `pip`

## Установка и запуск через uv

1. Создайте виртуальное окружение и активируйте его:

**Linux/macOS:**

```bash
uv venv 
source .venv/bin/activate
```

**Windows (PowerShell):**

```powershell
uv venv hangman_env
.\hangman_env\Scripts\Activate.ps1
```

2. Установите пакет в режиме разработки:

```bash
pip install . -e
```

3. Запустите игру:

```bash
hangman
```

## Установка и запуск через pip

1. Установите пакет напрямую (если он опубликован в PyPI) или локально:

**Локально:**

```bash
pip install -e .
```

2. Запустите игру:

```bash
hangman
```

## Требования

* Python 3.13+
* `uv` (только если используете метод через uv)
* Кроссплатформенность: Linux и Windows поддерживаются

## Пример игры

```
Выберите язык: [en/ru] ru
Выберите уровень сложности: [easy/medium/hard] easy

_ _ _ _ _
```

Игрок угадывает буквы, пока не угадает слово или не проиграет.

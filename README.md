# Hangman (Виселица)

Консольная игра «Виселица» с возможностью выбора уровня сложности и языка.

## Возможности

* Выбор уровня сложности: **легкий**, **средний**, **сложный**
* Выбор языка словаря: **английский** / **русский**

## Установка и запуск

1. Создайте виртуальное окружение и активируйте его:

**Linux/macOS:**

```bash
uv venv 
source .venv/bin/activate
```

**Windows (PowerShell):**

```powershell
uv venv
.venv\Scripts\Activate.ps1
```

2. Установите пакет в режиме разработки:

```bash
pip install . -e
```

3. Запустите игру:

```bash
uv run hangman
```

Теперь игру можно запускать так:
```bash
hangman
```
